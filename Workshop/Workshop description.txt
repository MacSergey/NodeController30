[code][h1]Overview[/h1]
[list][h2]This mod allows you change node type and shape[/h2][/list]
[/code]

This is renewal version of original [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2085403475]Node Controller[/url] by [url=https://steamcommunity.com/profiles/76561198109315306/myworkshopfiles/]kian.zarrin[/url] with improved UI/UX.
The new UI allows you to make the perfect intersection with just a few clicks.

Node Controller Renewal supports importing original Node Controller data, but not completely. Some intersections may require manual adjustment. 
To start use Node Controller Renewal you should disable original Node Controller and subscribe to [url=https://steamcommunity.com/workshop/filedetails/?id=2040656402]Harmony[/url]
[code]
[h3][b]WARNING[/b]: If you import data from original Node Controller and make new save, you will no longer be able to use the original Node Controller on this map.[/h3]
[/code]

[HR][/HR] 
[h1]What’s Included[/h1]
[list] 
[*] Change node type and shape.
[*] Make intersection sloped.
[*] Change segment ends offset, shift, rotate, twist, slope and stretch.
[*] Alignment of roads with each other.
[*] Add Crosswalks and UTurns.
[*] Hide crosswalk markings (required [url=https://steamcommunity.com/workshop/filedetails/?id=1934023593]Hide Crosswalk mod[/url])
[*] Move It integration.
[*] Save data in intersection assets.
[*] Translation into languages: Chinese, Dutch, Finnish, French, German, Japanese, Polish, Portuguese, Russian, Spanish
[/list]

[HR][/HR] 
[h1]How activate[/h1]
[olist]
[*] Press Ctrl+N (You can change binding shortcut in mod settings)
[*] Open Road panel and press button [img]https://i.imgur.com/Itw4p0H.png[/img]

[*] Use [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2255219025]UnifiedUI[/url] mod
[/olist]
